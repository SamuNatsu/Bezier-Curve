# Lib
Needs SFML for rendering. (https://www.sfml-dev.org/index.php)
# Build
```bash
$ make .
```
# Usage
**Right click empty area:** Create a new node

**Right click a node:** Delete the node

**Left click & drag a node:** Drag a node
# Limitation
20 nodes max
